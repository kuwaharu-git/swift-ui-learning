# SwiftUI ユーザー入力コンポーネント サンプルアプリ

このディレクトリには、SwiftUIのユーザー入力コンポーネントを実際に動作させて学べるサンプルアプリが含まれています。

## ファイル構成

- `UserInputExamplesApp.swift` - 完全なサンプルアプリ（すべての入力コンポーネントの例を含む）

## 含まれるコンポーネント例

### 1. TextField（テキスト入力）
- 基本的なテキスト入力
- パスワード入力（SecureField）
- メールアドレス入力（適切なキーボードタイプ設定）
- 数値入力
- 複数行テキスト入力（TextEditor）

### 2. Toggle（チェックボックス）
- 基本的なToggle
- 設定メニュー風のToggleリスト
- カスタムチェックボックススタイル

### 3. Picker（選択メニュー）
- セグメントスタイル（ラジオボタン風）
- ホイールスタイル
- メニュースタイル（ドロップダウン）
- カスタムラジオボタン

### 4. DatePicker（日付選択）
- 基本的な日付選択
- 時刻選択
- 日付範囲制限
- 過去の日付のみ選択可能

### 5. Slider（スライダー）
- 基本的なSlider
- カスタマイズされたSlider（色、アイコン付き）
- 温度設定Slider
- 価格範囲Slider

### 6. Stepper（ステッパー）
- 基本的なStepper
- カスタムStepper（温度設定）
- フォントサイズStepper
- 参加者数Stepper

### 7. 実践的なフォーム例
- **ユーザー登録フォーム**: すべての入力コンポーネントを組み合わせた実用的なフォーム
- **設定画面**: アプリの設定画面のような実用例

## アプリの使い方

### Xcodeでの実行方法

1. Xcodeで新しいSwiftUIプロジェクトを作成
2. `UserInputExamplesApp.swift`の内容を`ContentView.swift`に貼り付け
3. ビルドして実行

### 各画面の説明

#### メイン画面
- すべての入力コンポーネントへのナビゲーションメニュー
- 基本的なコンポーネントと高度なコンポーネントに分類
- 実践例も含む

#### TextField Examples
- さまざまなテキスト入力の例
- キーボードタイプの設定例
- 入力制限や自動補正の設定例

#### Toggle Examples
- オン/オフの状態管理
- 複数のToggleを組み合わせた設定画面
- カスタムスタイルの実装例

#### Picker Examples
- 異なるピッカースタイルの比較
- ラジオボタン風の実装
- ドロップダウンメニューの実装

#### DatePicker Examples
- さまざまな日付・時刻選択の方法
- 日付範囲の制限方法
- 日本語フォーマットでの表示

#### Slider Examples
- 値の範囲選択
- カスタマイズされた見た目
- 実用的な使用例（音量、明るさなど）

#### Stepper Examples
- 値の増減コントロール
- カスタムデザイン
- 実用的な使用例（数量、温度など）

#### ユーザー登録フォーム
- すべての入力コンポーネントを組み合わせた実践例
- バリデーション機能付き
- ユーザビリティを考慮した設計

#### 設定画面
- アプリの設定画面のような実用例
- 段階的な設定項目の配置
- 実際のアプリで使える設計パターン

## 学習のポイント

### 1. @Stateの使い方
すべての入力コンポーネントで`@State`を使って状態管理を行っています。

### 2. バインディング（$）
入力コンポーネントと状態変数の双方向バインディングを理解できます。

### 3. コンポーネントのカスタマイズ
標準のコンポーネントをカスタマイズする方法を学べます。

### 4. フォームのバリデーション
実用的なフォームでの入力チェック方法を参考にできます。

### 5. ユーザビリティの考慮
実際のアプリで使える設計パターンを学べます。

## 拡張のアイディア

このサンプルアプリをベースに、以下のような機能を追加してさらに学習を深められます：

- データの永続化（UserDefaults、Core Data、SwiftData）
- アニメーション効果の追加
- アクセシビリティ対応
- 多言語対応
- ダークモード対応
- iPad対応（レスポンシブデザイン）
